---

# Como baixar:

```
    $ git clone https://github.com/Giovani2912/Interface-Mercado-Livre

    $ cd Interface-Mercado-Livre

# Caso esteja utilizando o npm

    $ npm install

    $ npm start

# Caso esteja utilizando o yarn

    $ yarn add

    $ yarn start

```
